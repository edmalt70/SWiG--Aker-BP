Documents are located at this top level.
Matlab code to define and simulate the proposed protocol is located in the SimulatorCode subdirectory.
SimulatorCode/Doc/html/index.html is the main page for documentation of the simulator code
SimulatorCode/doxygen contain the files necessary to generate the documentation for the simulator cod
To regenerate the documentation:
1. A Windows computer is required
2. doxygen must be installed in C:\Program Files\doxygen
3. A perl interpreter must be installed and be on the system path (Strawberry Perl recommended)
